1.AR/VR Endless Runner Game

An immersive AR/VR-based endless runner game developed using Unity, designed to deliver an engaging and interactive 3D gaming experience. The project combines real-time player movement, obstacle avoidance, procedural environment generation, coin collection mechanics, and character customization features to create a dynamic endless gameplay system.

The game includes smooth controls, continuously spawning tracks and obstacles, collectible coins for score enhancement, and a character cloth-changing/customization system that improves player interaction and personalization. With increasing difficulty levels and optimized gameplay performance, the project demonstrates modern AR/VR game development concepts including real-time rendering, physics interaction, immersive environments, and interactive UI systems.

2.Features

Infinite endless runner gameplay

Real-time obstacle spawning

Character cloth changing/customization feature
Smooth player movement and controls

4.Objective

The objective of this project is to explore immersive AR/VR game development by building an interactive endless runner experience that combines engaging gameplay mechanics, character customization, collectible systems, and optimized real-time 3D environments.
